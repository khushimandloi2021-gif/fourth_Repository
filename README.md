# Head & Tail Coin Flip Game 🪙

A simple coin flip simulator built with Python!

## How it works
- Program randomly generates 1 or 2
- 1 = Head, 2 = Tail
- Result is displayed instantly

## Concepts used
- Random module
- if-else conditions
